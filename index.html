*User Guide

1. Masukkan angka pertama.

2. Pilih operator (Operator tidak akan muncul pada display namun akan ada alert jika operator belum/sudah dipilih).

3. Masukkan angka kedua.

4. Tekan tombol = untuk melihat hasil.

5. Tekan tombol CE untuk memulai operasi baru agar tidak menemukan error.
